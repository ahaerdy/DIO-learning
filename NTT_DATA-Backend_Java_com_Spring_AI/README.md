Este repositório reúne os exercícios, projetos e anotações desenvolvidos por mim durante o [Bootcamp NTT DATA: Backend Java com Spring AI](https://web.dio.me/track/bootcamp-ntt-data-backend-java-com-spring-ai), oferecido pela plataforma DIO em parceria com a NTT DATA.

O conteúdo abrange desde os fundamentos modernos de Inteligência Artificial e sua aplicação no dia a dia do desenvolvedor, passando por uma revisão aprofundada de Java orientado a objetos, boas práticas de código, construção de APIs REST com Spring Boot, integração com banco de dados e segurança, até a publicação e monitoramento de soluções de back-end na nuvem com Microsoft Azure.

## Tecnologias e Tópicos Abordados

Durante o Bootcamp, os principais conhecimentos praticados são:

- Inteligência Artificial aplicada ao desenvolvimento: Machine Learning, LLMs, IA Generativa e Agentes
- GitHub Copilot para aumento de produtividade no desenvolvimento
- Programação Orientada a Objetos avançada com Java (abstração, encapsulamento, herança, polimorfismo)
- Collections, Interfaces, Lambdas e Streams em Java
- Boas práticas: Clean Code, princípios SOLID, logging e tratamento de exceções
- Gerenciamento de dependências com Maven e Gradle
- Anotações em Java e geração de código inteligente
- Design Patterns clássicos (GoF) aplicados com Spring Framework
- APIs REST com Spring Boot e Spring Web
- Persistência de dados com Spring Data JPA e banco de dados relacional
- Segurança em APIs com Spring Security (JWT, OAuth2)
- Integração com APIs externas usando Spring Cloud OpenFeign
- Spring AI para construção de aplicações back-end com IA generativa
- Publicação, operação e monitoramento de aplicações no Microsoft Azure
