# Landing page com menu hamburguer e morphing menu (css)

Este repositório contém o projeto desenvolvido durante o curso  **“Desenvolvendo um Menu Hamburguer e Morphing Menu com CSS Transitions”**,  ministrado por **Gabriela Pinheiro** na plataforma **DIO (Digital Innovation One)**.

O projeto tem como objetivo consolidar conceitos de **estruturação HTML**, **estilização com CSS**, **animações**, **transições** e **responsividade**, sem utilização de JavaScript. ....

---

## Objetivos do Projeto

- Estruturar uma landing page utilizando HTML5
- Criar um menu hamburguer animado utilizando apenas CSS
- Aplicar transições e animações com `transition` e `@keyframes`
- Utilizar o seletor `:checked` para controle de estado visual
- Construir cards interativos com efeitos de hover
- Desenvolver layout responsivo para desktop e dispositivos móveis

---

## Tecnologias Utilizadas

- HTML5  
- CSS3  
- Flexbox  
- CSS Transitions e Animations  
- Media Queries  

---

##  Detalhamento do Projeto

Os arquivos index.html e style.css estão comentados detalhadamente, bloco a bloco, com explicações sobre a função de cada estrutura HTML e a lógica visual aplicada no CSS. Esses comentários fazem parte do processo de validação do aprendizado, demonstrando compreensão dos conceitos abordados.

## Considerações Finais

O projeto demonstra como é possível criar interfaces modernas, animadas e responsivas utilizando apenas HTML e CSS, reforçando boas práticas de organização, semântica e experiência do usuário.

Ele serve como base sólida para projetos front-end mais avançados e para composição de portfólio profissional.
