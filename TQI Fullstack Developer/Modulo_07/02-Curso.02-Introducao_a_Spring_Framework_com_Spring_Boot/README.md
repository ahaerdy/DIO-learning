O cronograma de aprendizado para o curso de **SpringBoot Framework** abrange os seguintes tópicos fundamentais:

* **Spring Versus Java EE**: Entendimento das diferenças e posicionamento de cada tecnologia.
* **Conceitos de Spring Boot**: Definição e objetivos da ferramenta.
* **IoC / DI**: Exploração dos conceitos de Inversão de Controle e Injeção de Dependência.
* **Beans, Autowired e Scopes**: Componentes essenciais do gerenciamento de objetos no Spring.
* **Spring Data JPA**: Noções para interação e persistência de dados em bancos de dados.
