Este repositório reúne os exercícios, projetos e anotações desenvolvidos por mim durante o [Bootcamp Globant: Java & Spring Boot AI Developer](https://web.dio.me/track/globant-java-and-spring-boot-ai-developer), oferecido pela plataforma DIO em parceria com a Globant.

O conteúdo percorre desde a configuração do ambiente de desenvolvimento com Java e Git, passando pelos fundamentos da linguagem, Programação Orientada a Objetos, boas práticas, Design Patterns, persistência de dados com SQL, desenvolvimento de APIs REST com Spring Boot, segurança com Spring Security, até a construção de aplicações inteligentes com Inteligência Artificial e a prática de testes unitários com JUnit 5.

## Conteúdo do Repositório

Cada diretório contém:
- Exercícios resolvidos relacionados aos fundamentos e práticas do desenvolvimento back-end com Java
- Notas e explicações sobre conceitos de POO, boas práticas, Spring Boot e Spring AI

- Recursos de estudo complementares utilizados durante a formação

## Tecnologias e Tópicos Abordados

Durante o Bootcamp, os principais conhecimentos praticados são:

- Configuração do ambiente Java com IDEs e ferramentas de desenvolvimento
- Controle de versão com Git e GitHub
- Fundamentos da sintaxe e estruturas de controle em Java
- Logging e tratamento de exceções
- Gerenciamento de dependências com Maven e Gradle
- Programação Orientada a Objetos avançada (abstração, encapsulamento, herança, polimorfismo)
- Collections, Interfaces, Lambdas e Streams em Java
- Inteligência Artificial aplicada ao desenvolvimento: Machine Learning, LLMs, IA Generativa e Agentes
- GitHub Copilot e Engenharia de Prompts para produtividade no desenvolvimento
- Fundamentos de Banco de Dados Relacionais e SQL
- APIs REST com Spring Boot e Spring Web (documentação com Swagger)
- Segurança em APIs com Spring Security
- Boas práticas: Clean Code, princípios SOLID e Design Patterns (GoF)
- Testes unitários com JUnit 5
- Spring AI para construção de aplicações inteligentes com IA generativa
